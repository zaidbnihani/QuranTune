# 📖 مشغّل القرآن — تطبيق استماع ذكي

**مشغّل القرآن** تطبيق أندرويد مفتوح المصدر يتيح الاستماع إلى القرآن الكريم والأذكار بطريقة ذكية وقابلة للتخصيص. يعتمد نظام **البطاقات** لتنظيم ما يريد المستخدم سماعه، ويدعم التشغيل التلقائي عبر الإشعارات، والمزامنة بين الأجهزة عبر MQTT، والتحميل للاستماع بدون إنترنت.

---

## ✨ المميزات الرئيسية

### 🃏 نظام البطاقات
أنشئ بطاقات مخصصة لكل سورة أو ذكر تريد تشغيله بسرعة. لكل بطاقة عنوان، صورة، قارئ، وكلمة تشغيل تلقائي.

### 🔔 تشغيل تلقائي عبر الإشعارات
يراقب التطبيق إشعارات جهازك — عندما تصل إشعار يحتوي على كلمة مفتاحية محددة مسبقاً، يبدأ التشغيل تلقائياً دون الحاجة لفتح التطبيق.

### 📡 مزامنة بين الأجهزة (MQTT)
اربط جهازَين معاً عبر QR Code، وتحكّم في تشغيل القرآن على جهاز آخر في الوقت الفعلي. يُستخدم بروتوكول MQTT عبر HiveMQ للمزامنة الفورية.

### ⬇️ تحميل للاستماع أوفلاين
حمّل أي سورة أو جزء عم أو أذكار الصباح والمساء لتشغيلها لاحقاً بدون اتصال بالإنترنت.

### 🕌 دعم كامل للغة العربية (RTL)
واجهة مصممة بالكامل للعربية مع دعم RTL أصيل وإشعارات وتوست بالعربية.

### 🎙️ دعم مئات القراء
تدعم قاعدة بيانات القراء أكثر من 200 قارئ من mp3quran.net مع إمكانية إدخال رابط خادم مخصص.

### 🤖 ذكاء اصطناعي (Gemini AI)
مدمج مع Firebase AI (Gemini) لميزات ذكية قادمة أو الاستفسارات المتعلقة بالمحتوى.

## 🛠️ التقنيات المستخدمة

| التقنية | الغرض |
|---|---|
| **Kotlin + Jetpack Compose** | واجهة المستخدم الحديثة |
| **Media3 (ExoPlayer)** | تشغيل الصوت عالي الجودة |
| **Room Database** | تخزين البطاقات محلياً |

| **NotificationListenerService** | التشغيل التلقائي عبر الإشعارات |
| **Firebase AI (Gemini)** | الذكاء الاصطناعي |
| **ZXing / CameraX** | مسح QR Code للربط بين الأجهزة |
| **Retrofit + OkHttp** | جلب بيانات القراء والمقاطع |
| **DownloadManager** | تحميل الملفات للاستماع أوفلاين |

## 📲 التثبيت

### تحميل مباشر (APK)
حمّل آخر إصدار من مجلد [`APK_DOWNLOAD`](./APK_DOWNLOAD/) وثبّت الملف مباشرة على جهازك.

> **تأكد من تفعيل "تثبيت من مصادر غير معروفة"** في إعدادات الجهاز.

### البناء من المصدر

1. **استنسخ المستودع:**
   ```bash
   git clone https://github.com/zaidbnihani/QuranTune.git
   cd QuranTune
   ```

2. **أعد تسمية ملف البيئة:**
   ```bash
   cp .env.example .env
   ```

3. **أضف مفتاح Gemini API:**
   افتح `.env` وضع مفتاحك:
   ```
   GEMINI_API_KEY=your_actual_key_here
   ```
   احصل على مفتاح مجاني من [Google AI Studio](https://aistudio.google.com/).

4. **افتح المشروع في Android Studio وشغّله.**

**المتطلبات:**
- Android 7.0 (API 24) أو أعلى
- Android Studio Hedgehog أو أحدث

## 🚀 كيفية الاستخدام

### 1. إنشاء بطاقة
اضغط على زر **+** لإنشاء بطاقة جديدة. حدد:
- **العنوان** (اسم السورة أو الذكر)
- **السورة** (من 1 إلى 114، أو أذكار الصباح/المساء، أو جزء عم)
- **القارئ** من القائمة أو برابط مخصص
- **كلمة التشغيل التلقائي** — الكلمة التي إذا وصلت في إشعار تشغّل البطاقة تلقائياً
- **صورة** اختيارية

### 2. التشغيل التلقائي عبر الإشعارات
- فعّل صلاحية **الوصول إلى الإشعارات** من إعدادات النظام
- عندما يصل أي إشعار يحتوي على الكلمة المفتاحية للبطاقة، يبدأ التشغيل فوراً
- لإيقاف الصوت، أرسل إشعاراً يحتوي على كلمة "إيقاف" أو "stop"

### 3. ربط جهازين
- اضغط على أيقونة **الإعدادات** ثم **ربط الأجهزة**
- اعرض QR Code على الجهاز الأول، وامسحه بالجهاز الثاني
- بعد الربط، التشغيل والإيقاف يتزامنان بين الجهازين فوراً

### 4. التحميل الأوفلاين
اضغط طويلاً على أي بطاقة واختر **تحميل** لحفظ الملف الصوتي على جهازك.
